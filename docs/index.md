# Welcome to OpenIoT

欢迎来到OpenIoT团队主页!

本团队由福州大学自动化系的3名本科生发起，截止目前拥有18名正式成员。项目立足于福建省工业自动化控制技术与信息处理重点实验室，受到福州大学本科生科研训练计划基金的支持。

我们致力于探索和应用物联网技术，为实现智能化生活和自动化生产做出贡献，充分利用所学知识与技能，为社会带来福祉。我们期待能够与您共同探索物联网技术的无限魅力！
