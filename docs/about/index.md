# 团队概述

OpenIoT团队最初由自动化系的**蔡汉霖、胡家齐、李政**发起，截止目前拥有12名学生成员，3位团队导师。立足于福建省工业自动化控制技术与信息处理重点实验室，受到福州大学本科生科研训练计划基金的支持。
